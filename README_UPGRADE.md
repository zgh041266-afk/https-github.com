# 权盾AI - 商标侵权智能取证与文书生成系统

## 系统升级说明

本次升级将原有的律师函生成 demo 改造为完整的**对话式商标侵权取证与文书生成平台**。

---

## 新增功能

### 1. 对话式交互界面

- **ChatGPT 风格的聊天界面**：用户通过自然语言描述需求
- **快捷功能按钮**：类案检索、文书生成、证据指引
- **实时流式响应**：AI 回复实时显示
- **文件上传支持**：可上传商标注册证、侵权截图等材料

访问地址：`http://localhost:3000/chat`

### 2. 三类核心产物输出

#### 2.1 侵权线索汇总表
- **表格展示**：序号、平台、商品标题、侵权URL、侵权类型、相似度得分
- **多选操作**：批量选择线索生成证据包
- **导出功能**：
  - CSV 格式（兼容 Excel）
  - Excel 格式（.xlsx）

#### 2.2 标准证据包
- **内容组成**：
  - 全屏截图（PNG 格式）
  - 文件哈希值（SHA256）
  - 元数据 JSON（URL、抓取时间、平台、商家信息）
- **命名规则**：`证据包_{packageId}.zip`
- **用途**：直接提交电商平台投诉系统或公证处

#### 2.3 商标侵权律师函
- **五大标准章节**：
  1. 权利基础与法律依据
  2. 侵权行为事实认定
  3. 要求义务
  4. 后续法律行动
  5. 联系方式
- **格式支持**：在线预览、DOCX 下载、PDF 导出

## 用户使用流程

### Step 1：进入对话界面
访问 `http://localhost:3000/chat`，看到类似 ChatGPT 的聊天界面。

### Step 2：输入需求
**示例输入**：
```
我是名创优品的法务，商标注册号13604462，在淘宝发现'优宿优品'店铺卖包，
疑似商标侵权，帮我取证并生成律师函。
```

### Step 3：系统自动解析
系统通过 MCP 工具自动：
- 提取关键实体（品牌名、商标号、平台、关键词）
- 识别用户意图（证据收集、类案检索、文书生成）
- 并行执行多个任务

### Step 4：查看结果
右侧面板实时显示：
- **侵权线索汇总表**：发现的疑似侵权商品列表
- **证据包列表**：已生成的标准证据包
- **律师函预览**：AI 生成的完整律师函

### Step 5：导出文档
- 点击"导出 CSV/Excel"下载侵权线索表
- 点击"下载 ZIP"获取证据包
- 点击"下载 DOCX"保存律师函

## 技术架构

### 前端
- **对话界面**：`/src/app/chat/page.tsx`
- **消息组件**：`/src/components/ChatMessageItem.tsx`
- **侵权表格**：`/src/components/InfringementTable.tsx`
- **证据包列表**：`/src/components/EvidencePackageList.tsx`

### 后端
- **对话 API**：`/src/app/api/chat/route.ts`
  - 接收用户消息
  - 调用 MCP 工具解析意图
  - 并行执行任务（类案检索、证据收集）
  - 流式返回 GPT-5.1 生成的回复

### MCP Server
新增 3 个工具：
1. **parse_user_intent**：解析用户自然语言输入，提取意图和实体
2. **search_similar_cases**：根据品牌名或商标号检索相似判例
3. **simulate_evidence_collection**：模拟证据收集流程（实际部署时对接真实爬虫）

### 数据类型
新增类型定义（`/src/types/index.ts`）：
- `ChatMessage`：对话消息
- `UserIntent`：用户意图解析结果
- `InfringementClue`：侵权线索
- `EvidencePackage`：证据包
- `TaskStatus`：任务状态

## 示例对话

**用户**：
```
我是名创优品的法务，商标注册号13604462，在淘宝发现'优宿优品'店铺卖包，
疑似商标侵权，帮我取证并生成律师函。
```

**系统响应**：
1. 解析意图：证据收集 + 文书生成
2. 提取实体：品牌名=名创优品，商标号=13604462，平台=淘宝，关键词=优宿优品
3. 检索类案：匹配到 MINISO vs USUPSO 案例
4. 模拟取证：生成 2 条侵权线索
5. 生成律师函：基于 MINISO 案例模板

**输出产物**：
- 侵权线索汇总表（2 条记录）
- 标准证据包（ZIP 格式）
- 商标侵权律师函（DOCX 格式）

- **协议**：MCP (Model Context Protocol)
- **导出**：xlsx + docx + file-saver
